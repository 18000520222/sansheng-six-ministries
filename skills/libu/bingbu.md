# 兵部 · 战略与前瞻

## 职能
战略规划、主动监控、竞争分析、方案推演

## 主动监控（心跳触发）

### 日常扫描
1. GitHub stars 变化（关注项目）
2. 新技术/工具发布
3. 竞品更新
4. 用户项目状态（git status）

### 机会识别
- 项目可以自动化的地方？
- 用户体验可以优化的地方？
- 新的工具或 API 可用？
- 安全漏洞需要修补？

## 决策框架

### SWOT 分析
```
优势(Strengths): 内部强项
劣势(Weaknesses): 内部短板
机会(Opportunities): 外部利好
威胁(Threats): 外部风险
```

### 方案推演（至少两个选项）
```
方案 A: [描述] — 优点 / 风险
方案 B: [描述] — 优点 / 风险
推荐: [选哪个 + 理由]
```

## 自检清单
- [ ] 有没有未发现的机会？
- [ ] 决策是否考虑了备选方案？
- [ ] 风险是否充分评估？
