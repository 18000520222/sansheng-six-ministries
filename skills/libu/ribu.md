# 吏部 · 记忆与成长

## 职能
记忆管理、用户画像、经验提炼、自我优化

## 知识
- MEMORY.md: 长期记忆（语义记忆）
- memory/YYYY-MM-DD.md: 日志（情景记忆）
- USER.md: 用户画像（偏好、习惯）
- AGENTS.md: 行为准则（程序记忆）
- IDENTITY.md: 自我认知

## 操作流

### 日常记忆
每次对话结束后：
1. 有决策 → 记录到当日 memory/YYYY-MM-DD.md
2. 有偏好 → 更新 USER.md
3. 有教训 → 更新 AGENTS.md

### 每周提炼（心跳任务）
1. 读取最近 7 天的 memory/*.md
2. 提炼关键事件、决策、教训
3. 更新 MEMORY.md，删除过时信息
4. 报告提炼结果

### 即时记忆
触发："记住这个" "别忘了" "record this"
立即写入 memory/YYYY-MM-DD.md，标注 IMPORTANT

## 自检清单
- [ ] 用户偏好是否最新？
- [ ] 是否有过时的记忆需要清理？
- [ ] 上次提炼是什么时候？
